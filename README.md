# Kaushal Event Talks App

This is a simple, single-page website to display the schedule for a one-day technical event.

## Features

*   Displays a full-day schedule with talk titles, speakers, descriptions, and categories.
*   Includes a 1-hour lunch break.
*   Talks are separated by 10-minute transition periods.
*   Users can search for talks by category.

## Tech Stack

*   HTML
*   CSS
*   JavaScript

## Getting Started

To run the website locally, you can use a simple Python web server.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/kaushal9808/Kaushal-event-talks-app.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd Kaushal-event-talks-app
    ```

3.  **Start the Python web server:**
    ```bash
    python3 -m http.server
    ```

4.  **Open the website in your browser:**
    Open your web browser and go to [http://localhost:8000](http://localhost:8000).

## Deployment

The `index.html` file is a self-contained, serverless file. You can deploy it to any static hosting service, such as GitHub Pages, Netlify, or Vercel.
